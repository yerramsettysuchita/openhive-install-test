# openhive-install-test

This repository is a clean room proof that OpenHive installs and works on a brand new project that has never seen it before. It was created from scratch, and OpenHive was added using only the two secrets and the single workflow file described in the installation guide.

## What this proves

Within ninety seconds of opening the first issue, the Triage Agent posted a human classification with three specific clarifying questions. Within two minutes of opening the first pull request, the PR Review Agent caught a false coverage claim and a real logic bug in the diff. Within two minutes of pushing a requirements file with outdated packages, the Security Agent queried the OSV database, found fourteen vulnerabilities across two packages, asked Claude for a remediation plan, and opened a patch pull request that upgrades both packages to safe versions.
